kitstrap v0.2

　kitstrapを導入しましょう。

ダウンロードして使う

　kitstrapの本体をダウンロードして使用を開始する場合、下のボタンからダウンロードできます。zipファイルをプロジェクトのディレクトリに展開してください。/d以下にkitstrapの本体が入っています。不要であればdocsディレクトリは削除しても問題ありません。 ダウンロード 　CSSファイルが設置できたら、次のようにlinkタグを挿入します。これだけでkitstrapを使用する準備は完了です。正しくCSSを読み込めているか開発者ツールなどで確認するとよいでしょう。 

<link rel="stylesheet" href="./d/kitstrap.css">

CDNから読み込む

　kitstrapのGitHubリポジトリを参照する形で読み込むこともできます。
　この方法では、ディレクトリ内へのバンドルが不要になりますが、github.ioへのアクセス環境がないとロードできません。
　また、次の方法では最新のkitstrapへ自動でアップデートされます。更新の変更内容によってウェブページの見た目が変化する場合があります。 

<link rel="stylesheet" href="https://mtsgi.github.io/kitstrap/d/kitstrap.css">
