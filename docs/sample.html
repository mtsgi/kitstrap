これは何？

kitstrapのサンプルです。実際にkitstrapによって生成される要素の見た目を確認できます。

class tag kit-button

kitstrapでは、class="kit-button"を付与または<kit-button></kit-button>タグを使用することで、簡単にボタン要素を作成できます。 ボタン1 ボタン2 ボタン3 ボタン4
デフォルトのカラーはdodgerblueです

class tag kit-hyperlink

a要素やその他の要素に対して、class="kit-hyperlink"を付与または<kit-hl></kit-hl>タグを使用することで、ハイパーリンクのスタイルを設定できます。 リンクはこちらをクリック！更新
指のカーソルになります

class tag kit-badge

インライン要素に、class="kit-badge"を付与または<kit-badge></kit-badge>タグを使用することで、バッジ要素を生成することができます。 NEW!kitstrapへようこそ
デフォルトのカラーはorangeです

class カラークラス

kit-buttonやkit-badgeなどのクラスはclass="kit-badge deeppink"のように、スペースを空けて色の名前を指定することで、簡単に色を変えることができます。 orangeバッジ・ボタン
