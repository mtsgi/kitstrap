これは何？

kitstrapのサンプルです。実際にkitstrapによって生成される要素の見た目を確認できます。

class kit-button

kitstrapでは、class="kit-button"を付与することで、簡単にボタン要素を作成できます。
ボタン1 ボタン2 ボタン3
デフォルトのカラーはdodgerblueです

class kit-hyperlink

a要素やその他の要素に対して、class="kit-hyperlink"を付与することで、ハイパーリンクのスタイルを設定できます。
リンクはこちらをクリック
指のカーソルになります

class kit-badge

インライン要素に、class="kit-badge"を付与すれば、バッジ要素を生成することができます。
NEW!kitstrapへようこそ
デフォルトのカラーはorangeです

class カラークラス

kit-buttonやkit-badgeなどのクラスはclass="kit-badge deeppink"のように、スペースを空けて色の名前を指定することで、簡単に色を変えることができます。
