kitstrap v0.6

kitstrapドキュメントへようこそ！kitstrapは軽量のCSSフレームワークです！

使いはじめる

Design Language

kitstrapは、kitDesktopのデザイン言語に基づいています。

Modularized

kitstrapパッケージでは、フォントやカーソルの設定はモジュール化されています。

Clean CSS

kitstrapは、!important規則を使用しません。

# 📦 Usage

npmを利用して導入することができます。

npm i kitstrap
# or
yarn add kitstrap

kitstrap.cssを読み込むことでも、使い始めることができます。

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mtsgi/kitstrap@1.0/d/kitstrap.css">

リリース一覧から、任意のバージョンをダウンロードして使用可能です。

<link rel="stylesheet" href="./d/kitstrap.css">

導入方法の詳細はこちら
